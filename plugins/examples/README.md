# Plugin Example Templates

This directory contains copyable plugin templates used by the case-study docs:

- `whisper_oop_template`
- `index_worker_template`
- `md_builder_template`

These templates are nested under `plugins/examples/` on purpose so default plugin discovery ignores them.

Use them by copying one template directory to `plugins/<your_plugin_name>/`, then adjust `name`, routes, and command.
